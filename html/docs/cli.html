CLI

Install. 

npm install -g js-web-cli

Setup js-web project 

js-web init

There are several optional arguments.

babel - babel will give you a project using babel. run a babel project (node run.js).

template - gives you html/style/script file setup width injections.

eslint - gives you js-web default eslint file.

mysql - gives you a db.js file, ready to write your migrations.

js-web init babel template eslint mysql

If you want a custom setup, you can use the env command to only get the .env file

js-web env
