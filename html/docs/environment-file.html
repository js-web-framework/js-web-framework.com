.env

You can get the .env file template here or with the js-web-cli (js-web env)

mac

Sets the mysql socketPath = '/tmp/mysql.sock'

mac=false

port

Sets til express listen port.

port=8080

allowCrossDomain

Allowing cors calls.

allowCrossDomain=false

https settings

https_privkey=
https_cert=
https_fullchain=
https=false

mysql credentials

mysql_host=localhost
mysql_user=root
mysql_password=
mysql_database=
mysql_port=3306

soft delete

Adding where {deleted_field} = 1 to all queries

mysql_soft_delete=false
mysql_soft_delete_field=deleted

mysql_query_debug

console.log all queries. 

mysql_query_debug=false

Mail credentials

mail_host=smtp.gmail.com
mail_port=587
mail_ssl=false
mail_user=name@gmail.com
mail_password=mypassword
mail_from=from@mail.dk

app_key

This should be a random string string; the app key is used as the session key.

app_key=writeRandomString

github_secret=

google_secret=
