JS-Web framework

About

JS-Web is built to uncomplicate web development by offering a simple and elegant API.
The focus is to break down web development to what it is - exposing endpoints.

Getting started

Make sure you have Node (and NPM) installed

Install the CLI

npm install -g js-web-cli

Setup a project

If you are using eslint (js-web init babel template eslint)

npm init
npm install js-web --save
js-web init babel template
